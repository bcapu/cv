Tags basicos

El tag " html "

Es el tag inicial y final , que contine los tags "head" y "body". Ejemplo: <html> </html>

El tag " head "

Contine informacion que no se ve en la pagina. Ejemplo:<head> </head>

El tag " body "

Es donde se coloca lo que se muestra en la pagina. Ejemplo: <body> </body>

El tag " h1 "

Escribe algo en letra grande, disminuye a medida q el numero aumenta ejs h2 h3 h4. Ejemplo: <h1 </h1>

El tag " br "

Es para dejar espacio y continuar abajo.No necesita cerrarse.

el tag " hr "

Resumen css

Cuando se va a agregar css a un elemento html, hay q tener en cuenta q los elementos tienen un valores por default.

por ejemplo, "hr" tiene por default "border-style: inset;"

Como comenzar css external

hay q crear una carpeta en el mismo nivel que el index, y adentro abrir un file y colocar un nombre que termine con ".css"
luego hay q ir al index y agregar el tag <link> en el "head" va a aparecer el atributo "href:" hay q modificar el nombre para q el browser sepa donde esta.

Anatomia css syntax

selector { property : value ; }

selector: es el "who?" quien queremos modificar
property: es el "what?", que cosa queremos modificar
value: es el "how?" como queremos modificar, que resultado queremos

properties

font-size: es para modificar tamanho

border-radius:100%; es para modificar una "box" en un circulo

text-align:center; es para centrar un texto

font-famyly: sans-serif; es para modificar las letras , sans-serif es la mas comun. hay q ir a la pagina de google font , para conseguir letras
ahi va a haber un link que hay q poner en el head y tambien el font family esta ahi

selectors class e id

ademas del tag, se puede usar como selector "class" o "id" para modificar algo.con class se usa .nombre{} y con id se usa #nombre{}

pseudo-class

es algo q se puede agregar adelante de un elemento para q haga una funcion, comienzan con ":"

por ejemplo ":hover", que se activa cuando pasas el mouse sobre el objeto al q se lo pusiste

the box model

cada elemento, esta contenido dentro de una "box", y tiene diferentes partes para modificar
el "margin", "border", "padding" y el elemnento en si mismo

display property

hay 4 tipos de display para las "box": block, inline , block-inline , none block : ocupa todo el espacio horizontalmente, se puede modificar el ancho inline : solo ocupa el espacio del elemento entonces si pones otra cosa la pone al lado y no abajo como block,pero no se puede modificar el ancho inline-block : es como inline pero se puede modificar el ancho none : es para quitar el elemento de la pantalla

position

static,relative,absolute y fixed

static es la pocicion por default

relative es para mover la pocicion relativo a su pocicion anterior

absolute, es para mover la pocicion relativo a su "padre"

fixed es para q quede fijado a algun lugar y mantenerse siempre ahi

float y clear

float es para mover texto alrededor de un elemento
clear es para blockear q el texto tenga algo en la direccion seleccionada
