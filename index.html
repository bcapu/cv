Resumen html

El tag <html> </html>

Es el tag inicial y final , que contine los tags "head" y "body".

El tag <head> </head>

Contine informacion que no se ve en la pagina.

El tag <body> </body>

Es donde se coloca lo que se muestra en la pagina.

El tag <h1> </h1>

Escribe algo en letra grande, disminuye a medida q el numero aumenta ejs h2 h3 h4.

<br>

No necesita cerrarse.

Es para dejar espacio y continuar abajo.

<hr>

Escribe una linea horizontal.

Con el atributo "size" se modifica el ancho de la linea , y con el atributo "noshade" se quita el contorno.
ejm <hr size="3" noshade>.

<center> </center>

centra todo lo que esta en el medio

<p> </p>

Es para escribir un parrafo

<em< </em>

Es para dar enfasis en algo que escribis, tranforma la letra en italic tambien se puede usar "i" "/i" pero no es recomendado

<strong> </strong>

Es para marcar las palabras como algo importante, tranforma la letra en negra tambien se puede usar "b" "/b" que viene de bold , pero no es recomendado

<ul> </ul> y <ol> </ol>

"ul" significa unordered list ,es para listas que llevan un punto al comienzo de cada item.

"ol" significa ordered list ,es para listas que llevan un numero al comienzo de cada item.

<li> </li>

Significa lista, y es lo q vas a poner en cada item de una lista.

<img src="asdasd.png" alt="ejemplo">

img ,es para agregar imagenes

src= ,es "source" la fuente de donde se saca esa imagen

alt= , es informacion alternativa o adicional, normalmente descripcion de la imagen.

asdasd.png ,es la imagen donde esta ubicada, se puede poner la imagen en la misma carpeta del proyecto para no tener q agregar informacion adicional para buscar la ubicacion, para esto tiene q esta en el mismo nivel q el index , sino hay q poner la ubicacion completa de donde esta.

<a href="httm//asdasd.com"> </a>

Es para poner links a otras paginas, u otras partes de la pagina tambien.
